¿Qué es el Box Model?

Para acomodar y organizar cada elemento HTML, en CSS se basan en capas para separar cada selector en unidades de medidas como: rem, px, em, cm, mm, etc. Y por la cual puede dar maquetaciones con mas claridad.

En algunas ocasiones se pueden colapsar los margenes, pero si uno de los valores es mayor que el otro selector. Como estos dos botones, que uno de ellos tiene mayor margen que el otro.
